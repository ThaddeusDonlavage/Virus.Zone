Panels — Overview

These panels are the primary rule screens used by the Virtual Veil tabletop system. Each panel focuses a major area of play so the GM and players can find rules quickly.

Panel List (Quick Links)

  • Core Mechanics — Dice, rolls, and resolution flow.
  • Character Creation — Attributes, COMPs, and abilities.
  • Monsters & Items — Gear, loot, and creature stats.

Purpose

Make rules modular and browsable. Each panel is intended as a concise reference during play: short, scannable, and linked to examples.

Layout Notes

Use this page as an index. Panels use the same header/footer as the main site and share `styles.css` for visual consistency.
